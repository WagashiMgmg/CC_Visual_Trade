# AGENTS.md — トレードボット学習ログ

振り返りから抽出したルールを蓄積します。
振り返り全文は `/app/data/reflections/trade_{id}.md` に保存されています。

## 基本ルール（手動設定）
- 最大保有時間: 4時間（強制決済） ← `.env` の `POSITION_MAX_HOURS` で変更可
- 分析間隔: 1時間ごと ← `.env` の `CYCLE_INTERVAL_MINUTES` で変更可
- ポジションサイズ: $100 USD, レバレッジ 3x
- 複数時間足が矛盾する場合は HOLD 優先

## 学習済みルール
（振り返りから自動追加・更新・洗練された最大１０件のルール）

- **マクロトレンド優先**: 月足・週足・日足がすべて下降トレンドの場合、ロングエントリー禁止（短期RSI過売れシグナルは無効化）
- **週足RSI過売れはベア相場では下落継続シグナル**: 月足・日足が下降トレンド中の週足RSI<30は逆張りロング根拠にならない。むしろ「強い下落モメンタムが継続している」証拠として解釈し、ロングエントリーをさらに厳格に禁止する（「マクロトレンド優先」の明示的補足）
- **回復トレードの乗り遅れ防止**: 急落からの回復が60%以上進行している場合、追いかけロングは避ける
- **デッドキャットバウンス判定**: 日足SMA20が下向きで価格がSMA20の10%以内に接近した場合（下落トレンド中のSMA20タッチ）はロングエントリー禁止。「回復」ではなく「レジスタンスへの接触」として扱う
- **エグジェキューション一致確認**: AI判断サイドと実行サイドが一致しているか、エントリー前に必ず確認する。不一致が発覚した場合は即座にポジションをクローズして原因を調査する
- **5分足デスクロスで逆張りロング禁止**: SMA20がSMA50を下抜けた状態での逆張りロングはエントリー条件から除外する
- **フィードラグ耐性**: 最低期待利益 = 往復フィー合計の2倍以上（$100×3x×0.1%往復≈$0.30フィー → 最低$0.60以上の期待値が必要）
- **シグナル鮮度チェック（判断・実行ともに適用）**: RSI・ブレイクアウト等のシグナルは参照時間足の直近2本以内のものを使用する（例: 5分足分析なら10分以内、1分足なら2分以内のシグナルのみ有効）。数時間前のシグナルを現在のエントリー根拠に使用してはならない。さらに、AI判断から実際のエントリー実行までも同様の時間制限を適用する（5分足ベースの判断なら10分以内に執行、それを超えた場合は判断を無効として再評価する）
- **レジスタンス帯ロング禁止**: 1時間足で確認できる直近高値圏（±$200以内）でのロングエントリーは禁止。レジスタンスを明確に上抜けてから乗る
- **ショート高値乖離チェック（ブルフラッグ禁止含む）**: 急騰後のショートエントリーは、①急騰高値（直近1〜4時間内のピーク）からの既下落幅が1%を超えている場合、フィードラグ耐性ルール（最低期待利益$0.60以上）を満たすかを必ず再検証。満たさない場合はエントリー禁止。②さらに、急騰後3時間以上にわたって価格が急騰高値の2%以内でコンソリデーション（横ばい）している場合は「ブルフラッグ形成中」と判断し、1時間足デスクロス（SMA20 < SMA50 かつ下向き）またはコンソリデーション安値の明確な下抜けが確認されるまでショートエントリー禁止。時間経過した高値ピークを現在のレジスタンスとして流用してはならない
